Alpine.js 学习 #

Alpine.js 是一个轻量级的 JavaScript 框架，可以理解为 JavaScript 版本的 Tailwind。

Alpine.js 当前的版本是 v3，它包含 18 个指令、9 个属性和 3 个方法。分别是：

  • 指令

    • x-data
    • x-text
    • x-html
    • x-init
    • x-show
    • x-cloak
    • x-transition
    • x-if
    • x-for
    • x-bind
    • x-on
    • x-model
    • x-effect
    • x-ignore
    • x-ref
    • x-teleport
    • x-modelable
    • x-id

  • 魔法属性

    • $el
    • $refs
    • $store
    • $watch
    • $dispatch
    • $nextTick
    • $root
    • $data
    • $id

  • 全局变量

    • Alpine.data()
    • Alpine.store()
    • Alpine.bind()
